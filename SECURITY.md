# Security Policy

## Supported Versions

Security updates are provided for the latest released version of this package.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ |

Older versions are not guaranteed to receive security updates. Users are encouraged to upgrade to the latest release.

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please report it responsibly.

**Please do not open a public GitHub issue for security vulnerabilities.**

Instead, report the issue by emailing:

**support@moxso.com**

Include the following information where possible:

- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes or mitigations

### What to Expect

- You will receive an acknowledgment within **72 hours**.
- We will investigate the report and work on a fix if the vulnerability is confirmed.
- Once resolved, a patched release will be published and the vulnerability may be disclosed publicly.

We appreciate responsible disclosure and your efforts to improve the security of this project.
